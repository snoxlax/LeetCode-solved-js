## Run Locally

Clone the project

```bash
  git clone https://github.com/snoxlax/LeetCode-solved-js
```

Go to the project directory

```bash
  cd LeetCode-solved-js
```

Run the solution by name using node.js

```bash
  node solutionName.js
```
